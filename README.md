# JoH Cosmetics 💄

JoH Cosmetics ni web app ya kuuza bidhaa za vipodozi online.

## Features 🚀
- View products
- Add to cart
- WhatsApp ordering
- Firebase database
- Admin panel
- Image upload with Firebase Storage

## Technologies Used
- HTML
- CSS
- JavaScript
- Firebase
- Netlify

## Project Structure

/index.html
/style.css
/script.js
/README.md

## Firebase Setup 🔥

1. Create Firebase project
2. Enable Firestore Database
3. Enable Firebase Storage
4. Add Firebase config inside script.js

## Deploy 🌐

Project imehostiwa kutumia Netlify.
